---------------------------------------------------------------------------------------------------------
Shaders are made by using an amplify shader editor.
---------------------------------------------------------------------------------------------------------

You can modify shaders and workflow as you need.

Also, there are several ASE functions.

“GetMfxSingle” - main logic of shader.

“GetMfxAlbedoNormal” - logic of albedo and normal textures.

“GetMetallicSmoothness” - recreated function from Unity Standard shader, and returns metallic and smoothness values (Metallic Workflow).

“GetSpecularSmoothness” -  recreated function from Unity Standard shader, and returns specular and smoothness values (Specular Workflow).
